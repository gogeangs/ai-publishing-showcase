AI + Human

AI는 도구입니다.
어디에 쓸지가 중요합니다.

AI가 잘하는 것

  • CSS 변수 일관성 (24/25점)
  • BEM 네이밍 체계적 적용
  • 시맨틱 HTML 구조화
  • Figma 에셋 자동 추출 (30개)
  • 기본 인터랙션 구현
  • 섹션별 코드 주석 정리
  • 빠른 초안 생성 (30분)

사람이 해야 하는 것

  • 다중 반응형 브레이크포인트 (10개+)
  • Swiper, GSAP 라이브러리 연동
  • 비디오 배경 / 복잡한 애니메이션
  • 서버 연동 (AJAX, reCAPTCHA)
  • 다중 폰트 / 커스텀 서체
  • 브라우저 호환성 테스트
  • 접근성(WCAG) 최종 검수

AI가 70-75%를 만들고, 사람이 나머지 25-30%를 완성합니다.
