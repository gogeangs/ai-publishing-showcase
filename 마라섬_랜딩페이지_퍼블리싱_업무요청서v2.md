## 4. 인터랙션 동작

> 정적 시안에서 알 수 없는 동작을 명시합니다.
> 기재하지 않은 인터랙션은 AI가 임의로 판단하거나 구현하지 않습니다.

### 4-1. 필수 인터랙션

| 위치 | 동작 | 상세 |
|------|------|------|
| GNB | 스크롤 시 배경 변경 | 100px 이상 스크롤 시 배경색 적용 + box-shadow |
| GNB | 메뉴 클릭 → 섹션 이동 | smooth scroll |
| 모바일 GNB | 햄버거 메뉴 열기/닫기 | 풀스크린 오버레이, body 스크롤 잠금 |
| 메뉴경쟁력 | 메뉴 슬라이더 | 좌우 화살표로 메뉴 이미지 슬라이드, 가운데 메뉴가 확대 표시, 하단에 활성 메뉴명 표시 |

| 창업문의 폼 | 토글 UI | 성별(남/여), 상가소지여부(네/아니오), 창업형태(신규/업종변경) — 라디오 버튼 스타일 토글 |

| 하단 문의바 | 간단 폼 유효성 | 이름/연락처 필수, 동의 체크 필수 |
| top버튼 | 스크롤 투 탑 | 클릭 시 페이지 최상단으로 smooth scroll, 일정 스크롤 이후에만 표시 |

## 7. 시안과 다르게 적용할 사항

## 8. 디자이너 미확인 사항 처리 기준

| 항목 | 처리 기준 |
|------|---------|
| 모바일 레이아웃 | 모바일 시안 없음. 768px 기준 1컬럼으로 자체 반응형 구현 |
| GNB 스크롤 시 배경 | 시안 배경색에 맞춰 구현 (MCP에서 색상 확인) |
| 하단 문의바 모바일 | 좁은 화면에서는 전화번호 + 신청 버튼만 표시 (입력 필드 숨김) |
| 도넛 차트 | CSS로 구현 (conic-gradient). SVG나 라이브러리 불필요 |
| "전액면제" 스탬프 | 이미지로 추출. MCP에서 못 가져오면 CSS로 빨간 스탬프 효과 |
| 모서리(corner) 장식 | MCP에서 SVG 추출. 못 가져오면 CSS border-image 또는 생략 |
| 오픈절차 아이콘 6개 | MCP에서 SVG 추출. 못 가져오면 텍스트 아이콘으로 대체 |
| 메뉴 슬라이더 | CSS transform + JS로 커스텀 구현. Swiper 없이 진행 |
| 히어로 3분할 배경 | 좌측(음식 콜라주 이미지), 중앙(크림/종이 텍스처 배경), 우측(매장 사진 반투명 오버레이) |
| 타겟 브라우저 | 모던 브라우저만 (IE 미지원) |
