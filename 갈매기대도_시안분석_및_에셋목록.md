## 4. 퍼블리싱 시 주의사항

1. **키카피 텍스트**: Figma 시안의 키카피가 모두 "100% 고객 만족 후기"로 표시됨. 업무요청서에 명시된 각 섹션별 실제 텍스트를 적용해야 함:
   - 차별점: "갈매기대도 만의 차별점"
   - 상차림: "갈매기대도의 상차림" / "한 끼 이상의 편안함을 담았습니다."
   - 고객후기: "100% 고객 만족 후기" / "꾸밈없는 정성과 태도가 손님들의 경험으로 남았습니다."
   - 창업절차: "창업절차" / "가맹문의부터 오픈 이후 운영까지 본사가 전 과정을 책임지고 지원합니다."
   - 창업비용: "창업비용" / "숨김없이, 과장 없이 실제 필요한 항목만 명확하게 안내드립니다."
   - 문의하기: "문의하기" / "30년 전통의 갈매기대도와 함께하실 점주님들을 기다립니다."

2. **아치형 텍스트**: KoreanTBSMR 폰트 사용 → 웹폰트 없음 → **반드시 SVG 통이미지로 Export**

3. **backdrop-filter**: Safari 호환을 위해 `-webkit-backdrop-filter` 함께 작성
   - 차별점 카드: `blur(4px)`
   - 배경 섹션: `blur(12px)`
   - 창업절차 카드: `blur(6px)`

4. **탭 활성 그라디언트**: radial-gradient 사용
   ```css
   background: radial-gradient(ellipse at center, #0f976b 0%, #087d57 50%, #006343 100%);
   ```

5. **하단 문의 바**: 동일 컴포넌트가 메인 비주얼 하단과 푸터 하단에 2회 등장 — sticky 동작 여부 별도 확인 필요
